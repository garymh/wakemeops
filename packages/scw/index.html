    scw

    Command-line interface for Scaleway

    Scaleway CLI is a tool to help you pilot your Scaleway infrastructure directly from your terminal.

    devops

    https://github.com/scaleway/scaleway-cli

    arm64 amd64

    Installation

    1. Add WakeMeOps repository

    curl -sSL https://raw.githubusercontent.com/upciti/wakemeops/main/assets/install_repository | sudo bash

    2. Install scw

    sudo apt install scw

    Snippets

    FROM wakemeops/minideb:bullseye

RUN install_packages \
    scw=2.16.1*

USER 1001

    - name: Install dependencies
  uses: upciti/wakemeops-action@v1
  with:
    packages: |
      scw=2.16.1*

    Blueprints

    Debian packages listed on this page are generated from op2deb YAML blueprints. Blueprints for scw are versioned here.

    Click here to see scw ops2deb blueprints
    https://github.com/upciti/wakemeops/blob/main/blueprints/devops/scw/ops2deb.yml
    - name: scw
  version: 2.3.1
  homepage: https://github.com/scaleway/scaleway-cli
  summary: command-line interface for Scaleway
  description: |-
    Scaleway CLI is a tool to help you pilot your Scaleway infrastructure
    directly from your terminal.
  fetch: https://github.com/scaleway/scaleway-cli/releases/download/v{{version}}/scw-{{version}}-linux-x86_64
  script:
    - install -m 755 scw-{{version}}-linux-x86_64 {{src}}/usr/bin/scw
